# SIKAL YOPER - WhatsApp Recovery System

Website asisten otomatis untuk membantu memulihkan akun WhatsApp yang terblokir secara permanen dengan enkapsulasi draf email ke Support WhatsApp via Gmail.

## 🚀 Fitur
- **Premium UI:** Desain gelap futuristik dengan efek kelopak bunga musim gugur bergerak.
- **Auto Sync:** Sinkronisasi nomor HP ke draf pesan secara *real-time*.
- **Client-Side Security:** Filter otomatis untuk mencegah input huruf/simbol ilegal pada nomor HP.
- **Responsive Layout:** Optimal dibuka di PC maupun Handphone (Android/iOS).

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom Animations)
- Vanilla JavaScript (Client Intent)

*Hosted premium on Vercel.*
